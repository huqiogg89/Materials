# Анализ проектов Цифрового Шелкового пути (DSR) в регионе АСЕАН

Данный проект представляет собой систему сбора, обработки и визуализации данных по технологической экспансии Китая в странах Юго-Восточной Азии (АСЕАН) в период с 2000 по 2025 гг.

## Структура проекта

### Данные
- `Таблица_ЦШП.csv` — основная база данных проектов (85 записей).
- `Таблица_ЦШП.xlsx` — версия таблицы в формате Excel.
- `data_rows.py` — исходный код базы данных в формате Python-словарей (источник для генерации CSV).

### Система парсинга и обработки
- `extractor.py` — скрипт для извлечения ссылок из библиографического списка и проверки их доступности.
- `export_to_excel.py` — утилита для конвертации данных в формат Excel с форматированием.

### Визуализация
- `plot_results.py` — базовый скрипт визуализации трендов по типам технологий.
- `plot_countries_by_year.py` — анализ активности по странам (линейный график).
- `plot_companies_by_year.py` — анализ активности по ключевым китайским компаниям (Alibaba, Huawei, ZTE и др.).

### Библиография
- `Источники для таблицы.txt` — полный список источников (145 позиций).
- `Новая библиография для Antigravity.txt` — оформленный список литературы.

## Результаты (Графики)
- `tech_trends_chart.png` — динамика по категориям технологий.
- `country_trends_chart.png` — активность по странам региона.
- `company_trends_chart.png` — лидеры среди китайских тех-гигантов.

## Установка и запуск

1. Убедитесь, что у вас установлен Python 3.8+.
2. Установите необходимые библиотеки:
   ```bash
   pip install -r requirements.txt
   ```
3. Для генерации графиков запустите соответствующий скрипт:
   ```bash
   python plot_companies_by_year.py
   ```
